Compass

  • Stylesheet Entwicklungswerkzeug
  • Support vieler CSS Frameworks
  • Generiert automatisch Stylesheets
  • URL handling
  • http://github.com/chriseppstein/compass
  • Einfach als gem installierbar

config.rb

              # Require any additional compass plugins here.
              project_type = :stand_alone
              # Set this to the root of your project when deployed:
              http_path = ""
              css_dir = "stylesheets"
              sass_dir = "src"
              images_dir = "images"
              http_images_path = "../images/"
              output_style = :compact
